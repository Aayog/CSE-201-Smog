# CSE-201-Smog
Meta-app to catalog games developed by Nozzle Co.










## Usage for now
### run the php server for the backend 
php -S localhost:8000
### then run angular
ng serve
if ng serve gives an error message try:
<ul>
<li> removing the node_modules directory </li>
<li> npm install</li>
<li>  ng update</li>
<li>  npm update</li>
<li> ng serve</li>
</ul>
